### 1. 設備準備：開啟「USB 偵錯」
如果您使用的是實體 Android 手機，請先完成以下設定（若使用模擬器通常預設已開啟，可跳過此步）：
1. 進入手機的 **「設定」** > **「關於手機」** > **「軟體資訊」**。
2. 找到 **「版本號碼 (Build number)」**，連續快速點擊 7 次，直到畫面提示「開發人員模式已啟用」。
3. 回到設定的首頁，滑到最底點選新增的 **「開發人員選項」**。
4. 找到並開啟 **「USB 偵錯」** 功能。
5. 用傳輸線將手機連上電腦。此時手機若跳出「是否允許 USB 偵錯？」的授權視窗，請勾選「一律允許」並點擊確定。



### 2. 正確的啟動時機與步驟
為了讓機器人的視覺辨識能順利運作，請務必按照以下順序啟動：
1. **進入遊戲盤面**：在手機或模擬器上打開數獨 App，並**點擊進入任意關卡**。（⚠️ 非常重要：請確保當前畫面停留在看得到 9x9 完整格子的下棋畫面，不要停留在遊戲大廳或選單區）。
2. **開啟外掛**：雙擊打開電腦上的 `SudokuBot.exe`。
3. **一鍵開跑**：在彈出的控制面板中，設定您想要自動掛機的局數，點擊「▶️ 開始」，就可以了！
